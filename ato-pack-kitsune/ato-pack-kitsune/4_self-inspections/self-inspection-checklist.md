# Self-Inspection Checklist (Fictional)

## Monthly
- [ ] Access review complete and stored (AC-2)
- [ ] Patch compliance report stored (SI-2)
- [ ] Vulnerability scan summary stored
- [ ] POA&M reviewed; overdue items escalated
- [ ] Logging review record stored (AU-2)

## Quarterly
- [ ] SSP deltas reviewed (major changes reflected)
- [ ] Baseline config reviewed (CM-2)
- [ ] IR tabletop or review performed

## Notes
This checklist is meant to be simple, repeatable, and reviewer-friendly.
